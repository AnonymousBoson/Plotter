Plotter
=======

Generic plotter

To run:
  * Load some CMSSW environment with SAMADhi installed
  * Edit the plots you want in ```plots/myplots.json```
  * Edit the samples you want in your plots in ```samples/mysamples.json```
  * Run with ```./runPlots.py -s samples/mysamples.json -p plots/myplots.json --intL <XX>```
